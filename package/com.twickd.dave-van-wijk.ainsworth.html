Ainsworth 🧙

The Connecting Experience You Deserve

Don't own Airpods, but still want the premium connecting experience? Ainsworth got you covered! Ainsworth allows you to add one of the 3 popup styles to all your bluetooth devices, and wired headphones. It does this in an extremely elegant and modern way. You can also configure each popup to your hearts content. (see screenshots)

3 Different Styles

Default The default popup aims to replicate the normal Airpods popup, if you like a fancy, clean popup, this is the one for you.
Pencil The pencil banner is heavily inspired by the Apple Pencil banner, if you prefer minimal and cleanness over anything, this popup will suit you perfect.
Modern If those ultra-clean things aren't for you, and you want all the bells and whistles, this is made just for you. This popup has 6 different actions:

  1. Volume Up
  2. Volume Down
  3. Play/Pause
  4. Siri
  5. Youtube
  6. Phone

Configure To your Liking

You can configure each device separately, the included options are:

  • Enabling/Disabling the popup
  • Selecting the app to open
  • The time that it takes to automatically dismiss the popup
  • Enable/Disable the background blur of the popup
  • Change the image of your device (only for popup 1 and 3)

Version 1.2

A Modern Connecting Experience For a Modern Jailbreak: Ainsworth v1.2

Ainsworth 1.2 is here, and with it it brings a lot of big and small changes:

  • Added ability to connect to all Bluetooth devices, not just audio ones. This includes for example Game-Controllers and Keyboards.
  • Adding devices is now much much easier before with the new way of adding: instead of having to connect each device, it automaticly pulls up a list of each previously connected bluetooth device (from the Settings -> Bluetooth -> Availible Devices list)
  • You can now remove devices
  • Preferences now sync instant and don't need a respring
  • The default view now looks better in darkmode, thanks to a background color change of the done button.
  • Improved the animation smoothness to closer match the original
  • Added haptic feedback to the buttons
  • Made app icon in the Modern view higher quality
  • Improved DRM

Thanks for using Ainsworth, and have fun with the new Unc0ver jailbreak ✌️
