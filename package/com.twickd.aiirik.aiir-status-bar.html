Aiir Status Bar Theme

Custom made status bar signal and wifi icons. Multiple styles to select from and mix and match to go along with your current setup.

New styles will be added in the future! But wait! Do you have an ideal for a status bar icon theme/style? Send me a message and we can see if we can bring it to life! Most of the testing was done on iPhone X using Snowboard. If you have any issues with any of the icons, contact me. Icons should work on all devices notch/notchless.

Requirements

Must have Snowboard and Snowboard Status bar Extension.
