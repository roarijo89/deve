Dynamic Wallpapers Generation!

Find out how to resolve incidents faster, with less friction.ads via Carbon

DynaWall is an amazing wallpaper app with a plethora of features. It strives to be a one stop shop for everything related to wallpapers.

Features

  • Create Dynamic Wallpapers from your Photo Library with a beautiful UI

  • Create Stock-like Live Wallpapers with videos from your Photo Library. (NEW in v2)

  • Create Gradient wallpapers- When you choose an image, you get a cool button that creates a beautiful looking gradient from the image and applies it to the other appearance. (NEW in v3.0)

  • Command line Mode: Provide a path to the Image files for light and dark and call dynawallcli and it’ll do the rest and set wallpapers for you. Just call dynawallcli --help for a nice overview of what you can do. (NEW in v3.0)

  • Smart Crop: Device Specific crop to create and set perfect wallpapers every single time. This uses a painstakingly handmade database for every single device on iOS 13 that makes Dynawall smarter than ever before. (NEW in v3.0)

  • Image Filters support. If you're looking for a monochrome look, head to settings and tap the Black and White filter option. Every image will now automatically be recolored to black and white. (NEW in v2.1)

  • Full iPad support: DynaWall works amazingly on iPads now with fullscreen, slide-over and split-view support. (NEW in v2.1)

(Note: iPads do not support live wallpapers out of the box. r/Jailbreak has guides to enable Live Wallpaper support)

  • Copy and reverse images button added: (NEW in v2.1)

    • If you have one image selected, a new copy button will appear, tap on it to copy the image to the other free slot. This can be used to make still wallpapers.
      • If you have both the images selected, the copy button will be a reverse button instead. You can swap the positions of the images.

  • Native iOS design

  • Magic Fill Feature: Load images from multiple online sources:

  • Load Images from Online Sources:

    • Unsplash:
      • Light Images are pulled from Nature, Mountains and Sea categories (Completely configurable, NEW in V2)
      • Dark Images are pulled from City and Night (Completely configurable, NEW in V2)
    • Images from Subreddits. (Yes, even memes.) (NEW in v2)
    • Load Bing's Daily wallpapers

  • Configure online sources, and determine where to load images from. (NEW in v2)

  • Don't like an online image? Mix and Match online images with images from your Library!

  • Image Pinning: Like one image but hate the other? Pin one image and cycle the other! Even change image sources from settings and choose images while having an image pinned! All with a beautiful design you expect from iOS. (NEW in v1.1)

Version 3.0

Dynawall v3.0

Changelog:

Dynawall v3.0 adds a whole bunch of features and bug fixes:

  1. Set wallpapers from inside Dynawall - No more fiddling around with settings and jumping between apps. With the brand new mode, you can set wallpapers right from inside Dynawall and choose if you want Parallax or not and not worry about anything else. Obviously, the settings for this look beautiful too. Sweet!
  2. Create Gradient wallpapers- When you choose an image, you get a cool button that creates a beautiful looking gradient from the image and applies it to the other appearance.
  3. A command line mode! Provide a path to the Image files for light and dark and call dynawallcli and it’ll do the rest and set wallpapers for you. Just call dynawallcli --help for a nice overview of what you can do. (Automatically changing wallpapers anyone wink wink? set up a cron job and a bash script to loop over images)
  4. Smarter App & Zoom in bug fix - Dynawall gets really really smart. It fixes the image zoom in issue by using a huge database of devices and the wallpaper sizes perfectly suited for them. Dynawall uses this to automatically size images and crop them without you lifting a finger. Let Dynawall do the hard work, while your experience remains the same — and you get better looking wallpapers. Win win! This feature will be optimized in the coming releases even further
  5. Magic Fill Crash fix - Something that I changed in 2.1 caused some people to instantly crash on tapping magic fill. A package I was using asynchronous image downloading was causing these crashes. I have switched over to a custom implementation and hopefully this version should fix that issue.
  6. iPad Crash Fix - A tiny crash fix that made Dynawall crash whenever any ActionSheet was brought up. That has also been fixed.

Version 2.1

DynaWall 2.1

Changelog

  • Introducing Image Filters. If you're looking for a monochrome look, head to settings and tap the Black and White filter option. Every image will now automatically be recolored to black and white.
  • Added button to copy images and reverse images:
    • If you have one image selected, a new copy button will appear, tap on it to copy the image to the other free slot. This can be used to make still wallpapers.
    • If you have both the images selected, the copy button will be a reverse button instead. You can swap the positions of the images.
  • Completely Redesigned Settings with custom made icons for every single option. This surprisingly took a lot of effort.
  • Much, much Faster image loading (Especially for Reddit images) with all network requests now being performed asynchronously.
  • DynaWall now checks for internet access before trying to load anything from the internet.
  • Proper iPad support. This supports DynaWall as a fullscreen app, slide-over and split-view apps. NOTE: iPad by default doesn't have Live Wallpaper support. People have reported success using this method (which I used on my SE to test live wallpapers too, as the SE doesn't support Live Wallpapers).
  • Haptics added to lock button.
  • App Theming You can now customize the tint color of the whole app. Make DynaWall fit in with your theme. (Thanks to Alderis, Color Picker).
  • I have added more ways to contact me.
  • Lots of tiny bug fixes:
    • Hopefully fixed the stupid haptic-engine-destroying button feedback (I test on an SE, so I don't even know how the haptics feel ( ; _ ; ) )
    • If you had a space in an Unsplash Category before, DynaWall got stuck permanently on loading. It should be fixed now.
    • Backend has been completely revamped now.
